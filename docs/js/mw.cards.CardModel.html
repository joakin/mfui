Class: CardModel

.cards.CardModel

new CardModel(attributes)

Model for an article It is the single source of truth about a Card, which is a representation of a wiki article. It emits a 'change' event when its attribute changes. A View can listen to this event and update the UI accordingly.
Parameters:
Name Type Description
attributes Object article data, such as title, url, etc. about an article
